# Gridventure Toolkit: World Generation System (Unity C#)

🚧 **Status: In Development (v0.1 - Initial Setup)**

A modular Unity world generation system designed for creating procedural environments using grid-based generation, auto-tiling, and Perlin noise.

This system is part of the **Gridventure Toolkit**, focused on building beginner-friendly, reusable systems for Unity game development.

## Overview

This project is currently under active development.

The goal is to provide a clean, extensible foundation for procedural world generation that can be easily integrated into both 2D Unity projects.

## Planned Features

* Grid-based world generation
* Perlin noise for terrain variation and biome distribution
* Auto-tiling system for seamless tile placement
* Configurable generation settings
* Modular architecture for easy extension

## Current Progress

* Project initialized
* Repository structure set up
* Planning system architecture and scope

## Design Goals

* Beginner-friendly and easy to understand
* Modular and reusable
* Clean separation of systems
* Consistent with other Gridventure Toolkit systems
* Scalable for more advanced generation systems (e.g., WFC)

## Future Systems

This system will connect with:

* Tile-Based Generation tools
* Auto-Tiling systems
* Advanced generation (WFC-based system)

## Notes

This is an early-stage system and is not yet ready for production use.

Documentation, demos, and features will be added as development progresses.
